# Transformer networks applied at time series: In-depth analysis of and visualization of the transformer attention
This repository contains various examples of transformer networks applied to time series scenarios. Some scenarios are generated with simple signals, like the sine wave, and there are examples with real-world scenarios, like the Kaggle store item dataset.

## Transformer examples:
- [In-depth analysis of transformer attention vs a random input](https://nbviewer.org/github/rodgdutra/neural_kaggle_store_item/blob/master/transformer_exemples/transformer_encoder_only_crazy.ipynb)
- [Transformer attention visualization with a sine wave input](https://github.com/rodgdutra/neural_kaggle_store_item/blob/master/transformer_exemples/transformer_sine_iterative_v2.ipynb)
- [Transformer applied in the Kaggle store item dataset](https://github.com/rodgdutra/neural_kaggle_store_item/blob/master/pytorch_transformer_enc_store_item.ipynb)

### LSTM and simple neural network example:
